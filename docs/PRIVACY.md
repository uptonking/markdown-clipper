Markdown Clipper processes browser data only to provide its core feature: copying links/ content as Markdown.

- Data processed:
    - Current tab title and URL
    - Selected text / link URL / image URL when you trigger copy actions
    - Open tab titles/URLs and tab group names when you use tab export features
    - Extension settings and custom templates stored in browser extension storage
- How data is used:
    - Generate Markdown and copy it to your clipboard.
- Data sharing:
    - We do not sell user data.
    - We do not share user data with third parties.
- Remote transmission:
    - The extension does not send your browsing data to our servers.
- Storage:
    - Settings/templates are stored in browser extension storage and remain until you
      change/remove them or uninstall the extension.

- Permissions:
    - activeTab, contextMenus, scripting: run user-invoked copy actions.
    - tabs, tabGroups (optional): export tabs/tab groups as Markdown.
    - storage: save settings/templates.
    - alarms: clear temporary badge status.
